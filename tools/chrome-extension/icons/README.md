# 图标文件说明

插件需要以下图标文件：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素  
- `icon128.png` - 128x128 像素

## 快速生成图标

你可以使用以下方法生成图标：

1. **在线工具**: 访问 https://www.favicon-generator.org/ 上传图片生成
2. **使用图标**: 可以使用任何 128x128 的图片，然后缩放生成其他尺寸
3. **临时方案**: 如果暂时没有图标，可以创建简单的纯色 PNG 图片

## 图标要求

- 格式: PNG
- 尺寸: 16x16, 48x48, 128x128
- 建议: 使用与视频/文字稿相关的图标，如 🎬 或 📝
